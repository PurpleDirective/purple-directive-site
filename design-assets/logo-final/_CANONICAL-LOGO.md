# Purple Directive — Canonical Logo (source of truth)

Last verified against the LIVE purpledirective.com header/hero on **2026-06-24**.
The current mark is the **two-tone faceted "D"** (deep-indigo body + royal-purple gem facet),
locked in the 2026-06-21 brand refresh. Everything older was archived — see
`_ARCHIVED-old-2026-06-24/`.

## Use THIS — vector marks live in `../../public/`
| File (in `1.Site/public/`) | When to use |
|---|---|
| **`pd-mark-grad.svg`** | **PRIMARY / default.** Full-color gradient faceted D. Use this whenever someone asks for "the logo" with no surface specified. |
| `pd-mark-dark.svg` | On DARK backgrounds only (nav, footer). Faceted D + thin `#b69bff` keyline. |
| `pd-mark-light.svg` | On LIGHT backgrounds only. Filled two-tone. **Do NOT hand this out as "the logo" by default.** |
| `pd-favicon.svg` | Favicon / app icon (filled two-tone on light rounded tile). |

> The recurring mistake: defaulting to **pd-mark-light**. Default is **pd-mark-grad**.
> Only reach for light/dark when the background surface is specified.
